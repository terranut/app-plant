## Variants

Las pantallas pueden contener más de un botón dedicado a diferentes acciones. Por este motivo es importante usar diferentes tipos o variantes de botón que nos permitan establecer una jerarquia visual.

<Canvas>
  <Story name="Variants" args={{ children: 'Button' }}>
    <Stack direction={'row'} spacing={'20px'}>
      <Button>Primary</Button>
      <Button variant="ghost">Ghost</Button>
      <Button variant="link">Link</Button>
    </Stack>
  </Story>
</Canvas>

## Sizes

A la hora de elegir el tamaño se debe tener en cuenta el dispositivo y la importancia de la acción.

## Disabled

Configura el botón para que se desactive cuando no se pueda utilizar. Se debe indicar el motivo por el cual el botón está desactivado al hacer hover sobre él.

## Loading

Para informar al usuario de que una acción se está ejecutando, el botón deberá pasar por el estado loading siempre que dicha acción requiera un tiempo de ejecución.

<Canvas>
  <Story name="Loading" args={{ children: 'Button' }}>

        Submit
      </Button>
    </Stack>
  </Story>
</Canvas>
